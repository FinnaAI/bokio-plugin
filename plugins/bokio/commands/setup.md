# Bokio Setup

Help the user configure their Bokio API credentials.

## Steps

1. **Explain what's needed:**
   - Company ID (UUID) - found in Bokio under Settings > Integrations
   - API Token - generated in Bokio under Settings > Integrations > API Access

2. **Ask for credentials** using AskUserQuestion or direct input

3. **Create the config file** at `~/.claude/bokio.local.md`:

```markdown
---
bokio_company_id: {user's company ID}
bokio_api_token: {user's API token}
---

# Bokio Configuration

Credentials for Bokio accounting integration.
Generated by /bokio:setup on {date}.
```

4. **Test the connection** by listing uploads:
```bash
curl -s "https://api.bokio.se/v1/companies/{COMPANY_ID}/uploads?page=1&pageSize=1" \
  -H "Authorization: Bearer {API_TOKEN}" | jq .totalItems
```

5. **Confirm success** or help troubleshoot if the API call fails.

## Where to get credentials

Direct the user to:
1. Log into https://app.bokio.se
2. Go to Settings (gear icon) > Integrations
3. Under "API Access", create a new token
4. Copy the Company ID from the URL or settings page
